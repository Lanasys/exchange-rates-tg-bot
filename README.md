<h3>Requirements:</h3>
<p>Versions are specified on which the bot was developed and tested</p>
<ul>
  <li>Python 3.10</li>
  <li>Aiogram 2.25</li>
  <li>uvloop 0.17.0 (not support at Windows)</li>
  <li>ujson 5.7.0</li>
  <li>cchardet 2.1.7</li>
  <li>aiodns 3.0.0</li>
  <li>aiohttp[speedups] 3.8.4</li>
  <li>Requests 2.29.0</li>
  <li>Pandas 2.0.1</li>
